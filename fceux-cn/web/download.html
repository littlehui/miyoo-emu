Downloads

The FCEUX team maintains two ports: SDL and Win32. The Win32 port has an extensive set of native gui amenities and tools. The SDL port supports many of the features of the Win32 build (Lua scripting, movie recording), but some of the Win32 features (debugger, TAS Editor) are exclusive to Windows. The SDL port should run in any UNIX-like OS (Linux/Solaris/BSD/OSX).

Network play in Win32 is not presently functional. Fixing it is a high priority.

Win32

  • Be sure to check out the fceux.chm file (Under Help -> Help... in the emulator main menu). We work pretty hard on keeping that up to date!
    • We also keep an up to date online version

SDL

  • Scons is required to build from source.
  • Requires SDL and libz
  • GTK2.0 and OpenGL are optional, but recommended.
  • The GTK GUI is enabled by default, but you can disable it by editing the SConstruct file.
  • For (somewhat outdated) instructions on installing on an Ubuntu or Debian system, check the how to at Ubuntuforums.

Download

  • FCEUX 2.2.3 win32 Binary
  • FCEUX 2.2.3 src

If you are working with a developer to fix an issue affecting you then this is where you will find your fixed build. It may potentially be out of date.

  • Interim Build $WCREV$ $WCDATE$

Source Code

  • SVN repository

FCEUX development is done commited to a subversion repository hosted at sourceforge. The last version of the source can be found there.

Compiling

  • Win32 uses MS Visual Studio 2010. Newer versions of visual studio should be able to build it, with a little trouble.
  • The SDL build uses Scons
  • Everything needed to compile the win32 version is included in SVN
  • The GTK port requires SDL and libz
