本版本整合了 CJS提供的汉化，金手指功能。感谢~

2. 追加汉化金手指功能。

ON=46,9

[飞机时无敌]
ON=E7A,3

[要塞内无限精力]
ON=DE7,FF;DE9,FF

[要塞内9束]
ON=DED,9;DEC,9

[GameInfo]

V2.2.3-M2
1. 修复了部分游戏双倍拉升显示不全的问题。
2. 添加slect + start 退出游戏。
3. 菜单的Load ROM现在改成Return Game

V2.2.3-M1
1. 支持震动。
2. 支持双倍硬件拉升，横扫描滤镜，网格扫描滤镜。

快捷键：
R2 菜单
L2 快进
R2 快速切换硬件拉升渲染方式
slect + start 退出游戏
L3 快速保存
R3 快速读取

R1 + 上 显示帧率
R1 + start 模拟器暂停
R1 + select 截图
